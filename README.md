# Password Strength Analyzer

A clean, modern password strength analyzer built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

- Dark-mode centered UI with a polished card layout.
- Password input field with a Show/Hide toggle.
- Real-time strength analysis based on length and character variety.
- Local top 10 common password check for insecure passwords.
- Dynamic color-coded strength bar.
- Rough brute-force crack-time estimate.
- Actionable optimization checklist to improve your password.

## Files

- `index.html` - Main HTML page with Tailwind UI.
- `script.js` - Password analysis logic and UI interaction.
- `README.md` - Project documentation.

## How to run

1. Open the `password-analyzer` folder in your browser.
2. Open `index.html` directly.
3. Type a password in the input field to see the strength score and suggestions.

## Notes

- This app is fully client-side and does not store any password data.
- The brute-force estimate is a rough approximation for demonstration purposes.
